# ng-conf-notes

### Install:
1.  Clone this repo: https://github.com/ChrisBauer/ng-conf-notes
2.  run `npm install`
3.  wait a while
4.  run `npm start`
5.  open your browser to http://localhost:8080


Seed project based on [angular2-webpack](https://github.com/preboot/angular2-webpack)
